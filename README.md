# rn-secure-keystore

 Secure key-value storage for React Native using Android Keystore and iOS Keychain

## Installation

```sh
npm install rn-secure-keystore
```

## Usage


```js
import { multiply } from 'rn-secure-keystore';

// ...

const result = await multiply(3, 7);
```
